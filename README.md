# WWE 2K26 Playbook — Public Read-Only Book

A public, read-only web view of the WWE 2K26 Playbook: champions, tournament
brackets, the Empire of Pain factions, and the full roster.

**Live page:** https://twitchtvkennysignguy9009.github.io/wwe2k26-book/

This repo contains **only the book data + a read-only viewer** — none of the
editing apps or their source. The full Playbook (the Electron apps you run on
stream) lives in a separate private repo.

## Updating it

When the roster, factions, or tournaments change, double-click
**`Update_Public_Book.bat`**. It copies the latest data from the live Playbook
and pushes it here; the public page refreshes within about a minute.
